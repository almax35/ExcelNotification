# ExcelNotificator
Многомодульное приложение для парсинга файлов xlsx и отправки уведомлений на почту
### Стек технологий: 
ExcelParser service: 
- Core: Springboot 3, Java 17
- Spring Mvc, Data Jpa
- DB: Postgres
- ORM: Hibernate
- Migration DB: Flyway
Notificator service:
- Core: Springboot 3, Java 17
- Spring Mvc, Data Jpa
- Mail
